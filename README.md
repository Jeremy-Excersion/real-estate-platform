## About Real Estate Platform

Real Estate Platform is an open-source project built on Laravel for teams in the real estate industry. This project will contain a frontend for visitors to view listings, save favorites, subscribe to notifications when new houses come on the market that meet their search criteria. It will also contain a backend where REALTORSÂ® can keep track of their leads, send them messages, and create saved searches for them.
